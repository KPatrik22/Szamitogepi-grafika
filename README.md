# Számítógépi grafika beadandó leírása
## Rövid leírás
- Egy fát látunk, amit meg lehet rázni. A rázással nő annak a valószínűsége, hogy lehullik-e egy-egy levele.
## Alapvető funkciók
- Fa rázása
- Kamera mozgatása
- Kamera forgatása
## Egyéb funkciók
- Fény állítása
- Kamera alaphelyzetbe állítása
- Használati útmutató
## Követelmények
### Alap
- Kamerakezelés
- Objektumok
- Animáció, mozgatás
- Textúrák
- Fények
- Használati útmutató
### Bónusz
- Köd és/vagy Eső
